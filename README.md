How to run the application:

1. Navigate to where you want to download it in  your terminal, and run 'git clone git@github.com:KjetilAlfredsen/IKT205_assignment_2"
2. Navigate into the IKT205_assignment_2 directory, and run "npm install"
3. use command "npx expo start"
4. Open the app on phone with the Expo Go app (some features may not work on the emulator, use 'npx expo start --tunnel' if you cannot open it on phone).
